FEATURES · 核心功能

一個平台，
讓商品成為可流通的票。

從建立到核銷，從贈送到分潤，愛票網提供商家完整的票證生命週期管理。不需自建系統，幾個步驟就能上線。

01 / TICKETING

商品一鍵電子票化

設定面額、有效期、使用條件、轉贈規則，系統自動生成獨立 QR Code 與序號，每一張票都可獨立追蹤。

02 / DELIVERY

多通路發送與贈送

透過 Email、簡訊、LINE、社群連結直送顧客手中。支援電子禮品卡的祝福訊息與排程寄送，送禮也是體驗。

03 / VERIFY

店端秒速核銷

店員以手機掃碼即可驗證使用，狀態即時更新，避免重複核銷與偽造票券。

★ PATENT · 專利申請中

顧客自我核銷

解決商家人力缺口的獨家機制，由顧客本人在合規條件下完成票券核銷，店員無須經手。了解更多 →

04 / ANALYTICS

銷售與行為分析

銷售熱度、核銷率、贈送網絡、過期回收，每一個數字背後都是下一波活動的機會。

05 / BRANDING

品牌客製化版型

票面色彩、字體、圖像、背景皆可自訂，每張票就是一張行走的品牌名片。

06 / INTEGRATION

API 與第三方系統串接

對接電商平台、POS、CRM、會員系統。從商品上架到核銷紀錄，資料雙向同步。針對連鎖品牌與大型零售業者，提供專屬技術窗口協助導入。

07 / SECURITY

金鑰加密與防偽機制

動態 QR Code 與時間戳簽章，每一次驗證都是新鑰匙，杜絕截圖、複製、轉售等風險。

08 / CAMPAIGN

活動與分銷管理

支援限時優惠、買就送、分潤合作、團購券，讓票券成為驅動成長的行銷工具。

PATENT PENDING · 專利申請中

讓顧客自己核銷，
把店員從櫃台解放。

尖峰時段人手不夠？外場、外送、無人櫃位無法掃碼？愛票網獨家研發「自我核銷」機制，顧客可在符合條件的時間與地點，由本人完成票券核銷，店家不必再為此分配人力，已申請專利保護。

  • 解決人力缺口　尖峰、深夜、無人化場域不再卡核銷流程，店員可專注於服務與出餐。
  • 多重防偽驗證　搭配時間戳、地理範圍、動態金鑰與單次有效碼，杜絕截圖、偽造、重複使用。
  • 店家可控規則　依商品類型彈性開關，可設定僅限店內、限定時段或先預約後自助核銷。
  • 即時對帳同步　顧客自助核銷後，後台即時更新狀態與報表，與店端掃碼核銷無差別。

TRUST · 銀行履約保證

每一張票券，
都有銀行為你撐腰。

愛票網已對接合作銀行，所有電子票券銷售款項自動交付信託，待顧客成功核銷後再撥款給商家。商家無需自行洽談銀行、無需準備信託押金，平台一站到位處理合規事項。

此機制讓消費者對購買行為更安心，也大幅降低中小商家進入電子票券市場的門檻。

合作銀行履約信託 款項信託、合規撥款

01

02
款項交付銀行信託 銷售金額自動進入合作銀行履約專戶，受監管保護。

FAQ · 常見問題

關於收費與合作

抽成只針對「成功核銷」的票券面額計收。例如顧客購買面額 NT$ 1,000 的電子禮券，使用 Pro 版（3% 抽成）的商家，在顧客核銷後，平台將收取 NT$ 30 作為交易服務費，剩餘金額由銀行履約專戶撥款給商家。未售出或未核銷的票券不會被收費。
費用組成共兩項：方案月費（依基礎版／Pro 版／商業版而定）以及交易抽成（5% / 3% / 1%）。銀行履約信託費、金流手續費均已含於抽成中，無額外收費。若有客製化開發、API 串接、私有部署等進階需求，將於合作洽談時另行報價並書面確認。
當顧客購買電子票券時，款項會自動進入愛票網與合作銀行設立的履約信託專戶，受銀行監管。當顧客完成核銷（履約義務確立）後，依約定週期撥款給商家。整個過程：
  • 商家不需自行對接銀行、不需準備信託押金
  • 消費者付款受到信託保障，商家若無法履約有撥付保護
  • 所有金流紀錄可於後台查閱

三個方案皆支援自我核銷。基礎版提供標準自我核銷模式；Pro 版可依商品開關功能、設定地理範圍與時段限制；商業版提供完整客製化規則引擎，可結合會員系統做更精細的核銷判斷。專利保護的多重防偽機制全方案一致適用。
標準結算週期為每月兩次（每月 15 日與月底），銀行於核銷確認後依週期撥款至商家指定帳戶，並提供完整對帳明細。商業版可協商更短的結算週期。
基礎版與 Pro 版從合約簽訂、銀行履約對接到首批商品上架，約 5–10 個工作日可完成。商業版因涉及 API 串接、客製化票面、系統整合等需求，依範圍評估，通常為 4–8 週。所有時程於合作洽談時提供明確時間表。
